## Techonologies
The technologies are used to make this site are :

- NEXT. JS
- Tailwind CSS
- React. JS
- React-Form-hook
- React-Portable-Text
- React-Icons
- Typescript
- Sanity CMS
- GROQ database
- Increamental Static Regeneration
- Server-side Rendering
 
## Features

- Content managed by Sanity CMS
- Regenerate the page after 60s 
- Commenting system
- Fast and smooth
- Well designed with Tailwind
